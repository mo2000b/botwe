# Botwe

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/mo2000b/botwe.git
   cd botwe
   ```

2. Install dependencies:
   ```bash
   cd project3
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. For production build:
   ```bash
   npm run build
   ```
